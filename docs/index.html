WiiFit Fitness App

by Luka Moleh

A comprehensive fitness application that helps users manage their workout routines, track their progress, and achieve their fitness goals. Built with modern web technologies, WiiFit offers a seamless experience for booking classes, scheduling personal training sessions, and managing fitness memberships.

Visit App View on GitHub

Key Features

  • User authentication and profile management
  • Group fitness class booking system
  • Personal trainer scheduling and availability
  • Membership plans with Stripe integration
  • Admin dashboard for class management
  • Responsive design for all devices

Technology Stack

  • Frontend: React + TypeScript + Vite + Tailwind CSS
  • Backend: Node.js + Express + PostgreSQL
  • Deployment: Vercel (Frontend) + Render (Backend)
  • CI/CD: GitHub Actions with automated testing
